# Analizador de Datos de Ventas

Proyecto de análisis de datos desarrollado en Python.

## ¿Qué hace?
- Carga y procesa datos de ventas
- Genera estadísticas descriptivas automáticamente
- Crea un gráfico de barras con las ventas por producto
- Exporta un reporte en formato Excel

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- OpenPyXL

## Archivos
- `python` — código principal del análisis
- `grafico_ventas.png` — gráfico generado automáticamente
- `reporte_ventas.xlsx` — reporte exportado en Excel
